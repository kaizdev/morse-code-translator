English <> Morse Code Translator

. -. --. .-.. .. ... .... / - --- / -- --- .-. ... . / -.-. --- -.. .

Instructions:

English words and letters are accepted for translation. Numbers and special characters are not translated.

Each english letter will be translated to morse code with a space representing a new letter. Each word will be separated using a forward slash (/)

Morse Code must be entered as periods, dashes, spaces or forward slash

Interesting facts about Morse Code:

  1. Samuel Morse invented Morse Code in the 1830s
  2. Morse Code was heavily used for secret transmissions during WWI and WWII
  3. SOS was chosen as a distress signal because it is easily understood and transmitted as Morse Code (3 dots, 3 dashes, 3 dots)
